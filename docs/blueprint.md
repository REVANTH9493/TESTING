# **App Name**: U-Learn

## Core Features:

- Course Listing: Display available courses with descriptions, instructors, and pricing.
- UPI Payment Integration: Enable UPI payments using a JSON configuration that opens payment apps on mobile.
- Mobile Payment Redirection: A 'Pay Now' button will extract payment URLs from the JSON to automatically redirect the user to their preferred payment application on mobile.
- Content Delivery: Securely serve course content such as videos, documents, and interactive exercises.
- User Authentication: Allow users to create accounts, log in, and manage their profile information.
- Personalized Learning Paths: AI powered tool will provide recommendations of new course to the user.

## Style Guidelines:

- Primary color: Vivid blue (#3498db) for trustworthiness and education.
- Background color: Light blue (#eaf2ff), creating a calm and inviting atmosphere.
- Accent color: Soft green (#2ecc71) for a sense of growth and progress.
- Headline font: 'Space Grotesk' sans-serif font for a modern, tech-forward feel (use 'Inter' for body).
- Body font: 'Inter', a grotesque-style sans-serif known for being readable and clean.
- Use flat, consistent icons for course categories and navigation elements.
- A clean, responsive layout with clear sections for course listings, payment options, and user profiles.
- Subtle transitions and animations to enhance user interaction, such as loading screens or button hover effects.
